# World Cup 2026

Static FIFA World Cup 2026 schedule, scores, points, bracket, favorites, and exports page for GitHub Pages.

The published page is expected at:

```text
https://numerate64.github.io/worldcup2026/
```

Data is polled hourly from `openfootball/worldcup.json` into `world-cup-2026.json`.
When the generated JSON changes, GitHub Actions commits the update and deploys GitHub Pages.
